Langtons Ant

Click here for the Repository!

Langton's ant is one of the classic cellular automata. It follows two simple rules.

- If the tile is black, turn it white, rotate clockwise 90 degrees and then move forward one.

- If the tile is white, turn it black, rotation anti-clockwise 90 degrees and then move forward one.

These two rules create this interesting pattern which results in what is called a "highway" in which the ant ends up in this repeating pattern which continues in a direction.
